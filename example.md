# 마크다운 사용 가이드

이 가이드는 Markdown Tree Editor에서 사용할 수 있는 마크다운 문법을 소개합니다.

## 📝 헤딩 (Headings)

문서의 제목과 부제목을 만들 때 사용합니다.

# 헤딩 1 (H1)
## 헤딩 2 (H2)
### 헤딩 3 (H3)

## ✏️ 텍스트 강조

**볼드 텍스트** - 별표 2개로 감싸기

*이탤릭 텍스트* - 별표 1개로 감싸기

**_볼드와 이탤릭_** - 조합도 가능합니다

## 💻 코드

### 인라인 코드

문장 안에 `코드`를 넣으려면 백틱으로 감쌉니다.

예: `const variable = 'value';`

### 코드 블록

여러 줄의 코드는 3개의 백틱으로 감싸고, 언어를 지정할 수 있습니다.

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
def calculate_sum(a, b):
    return a + b

result = calculate_sum(5, 3)
print(f"결과: {result}")
```

## 📋 리스트

### 순서 없는 리스트

- 항목 1
- 항목 2
- 항목 3

### 순서 있는 리스트

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목

## 💬 인용문 (Blockquote)

> 이것은 인용문입니다.
> 여러 줄도 가능합니다.

> 중요한 내용을 강조할 때 사용하세요!

## 💡 유용한 팁

- 트리 구조를 활용하여 문서를 체계적으로 구성하세요
- 드래그앤드롭으로 섹션의 순서를 쉽게 변경할 수 있습니다
- 미리보기 모드로 작성한 내용을 바로 확인하세요
- 버전 관리 기능을 활용하여 이전 내용을 복구할 수 있습니다
